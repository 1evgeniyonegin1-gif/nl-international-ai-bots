# Экспорт чата лидера

Сюда нужно положить экспорт чата вашего лидера/наставника из Telegram.

## Как сделать экспорт

1. Откройте Telegram Desktop (мобильная версия не поддерживает экспорт)
2. Откройте чат/канал лидера
3. Нажмите ⋮ (три точки) → **Export chat history**
4. Выберите:
   - Format: **JSON**
   - Media: можно отключить (не нужны)
   - Size limit: максимум
5. Нажмите **Export**
6. Скопируйте файл `result.json` в эту папку

## Структура файла

Telegram создаст файл примерно такого формата:

```json
{
  "name": "Название канала",
  "type": "public_channel",
  "id": 12345,
  "messages": [
    {
      "id": 1,
      "type": "message",
      "date": "2024-01-01T12:00:00",
      "text": "Текст сообщения"
    }
  ]
}
```

## Как это работает

1. Бот парсит все сообщения из `result.json`
2. Категоризирует их:
   - **product** — посты о продуктах
   - **motivation** — мотивационные посты
   - **business** — посты о бизнесе
   - **tips** — советы и лайфхаки
   - **news** — новости и акции
   - **success** — истории успеха
3. При генерации поста берёт **ТЕМУ** из соответствующей категории
4. Но пишет в **СВОЁМ стиле** (персоны Данила + эмоции)

## Важно!

- Бот берёт только **ТЕМЫ**, не копирует стиль лидера
- Стиль определяется системой персон (crazy, friend, rebel, expert, philosopher, tired)
- Если экспорта нет — используются встроенные темы
