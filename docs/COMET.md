## C - Context (Контекст)

### Что это за проект?

Система из двух AI-ботов для автоматизации бизнеса в NL International:

1. **AI-Куратор** - персональный ментор для партнёров
2. **AI-Контент-Менеджер** - автоматическая генерация контента для канала

### Проблемы, которые решаем

| Проблема | Как было | Как будет |
|----------|----------|-----------|
| Поддержка партнёров | Вручную, не 24/7 | AI отвечает мгновенно, круглосуточно |
| Создание контента | Часы на написание постов | AI генерирует за секунды |
| Обучение новичков | Забывают, теряются | Куратор ведёт и напоминает |
| Консультации по продуктам | Нужен эксперт | AI с базой знаний отвечает точно |
| Мотивация команды | Зависит от настроения | AI всегда позитивен и поддерживает |

### Целевая аудитория

- **Партнёры NL International** - получают поддержку от куратора
- **Администраторы** - управляют контентом через бота
- **Подписчики канала** - читают качественный контент

### Текущее состояние

| Компонент | Статус |
|-----------|--------|
| AI-Куратор | Работает на GigaChat |
| RAG база знаний | 5 документов проиндексировано |
| Контент-Менеджер | Код готов, нужен токен бота |
| Хостинг | Локально, нужен деплой 24/7 |

        ▼

          ▼

### Система промптов

#### Куратор - NutriМентор

- Эксперт-нутрициолог с 10-летним опытом
- Специализированные промпты по темам:
  - Управление весом
  - Энергия и работоспособность
  - Красота изнутри
  - Спортивное питание
  - Детокс и пищеварение
  - Работа с возражениями

#### Контент-Менеджер - CreativeNinja

- Копирайтер NL International
- 18 типов текстовых постов
- Контент-план на неделю (21 пост)
- Правила: живой тон, CTA, без гарантий

## E - Evaluation (Оценка)

### KPI для AI-Куратора

| Метрика | Как измеряем | Цель |
|---------|--------------|------|
| Время ответа | Логи (timestamp) | < 3 сек |
| Активные пользователи | Уникальные user_id/день | 50+ |
| Сообщений в день | COUNT в БД | 100+ |
| Retention | Возвращаются через 7 дней | > 60% |
| Ошибки AI | Жалобы, фидбек | < 5% |

### KPI для Контент-Менеджера

| Метрика | Как измеряем | Цель |
|---------|--------------|------|
| Постов в неделю | COUNT в БД | 21 |
| Одобрено сразу | approved / total | > 70% |
| Время модерации | approved_at - generated_at | < 5 мин |
| Просмотры поста | Telegram API (планируется) | 500+ |
| Реакции | Telegram API (планируется) | 20+ |

### KPI бизнеса

| Метрика | Как измеряем | Цель (3 мес) |
|---------|--------------|--------------|
| Новых партнёров | Регистрации через бота | +50 |
| Активность команды | Сообщения куратору | +200% |
| Рост канала | Подписчики | +300 |
| Экономия времени | Часы на рутину | -10 ч/нед |
| Стоимость системы | Расходы на хостинг | < $15/мес |

### Как собираем данные

1. **Логи ботов** - все действия записываются
2. **БД PostgreSQL** - статистика сообщений, постов
3. **Таблица AdminAction** - все действия модератора
4. **Ручной фидбек** - опросы партнёров
