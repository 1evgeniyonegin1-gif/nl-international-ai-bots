# Инструкция для партнёра: Как подключить своего AI-бота

**Версия:** 1.0
**Дата:** 21 января 2026

---

## Что ты получишь

После настройки у тебя будет:
- **Свой Telegram-бот** с AI-куратором
- **Автоматическая воронка продаж** для твоих клиентов
- **Публикация контента** в твою группу
- **Все ссылки** ведут на твой ID партнёра NL

## Шаг 1: Создание Telegram-бота

### 1.1 Открой @BotFather

1. В Telegram найди бота **@BotFather**
2. Нажми **Start** или отправь `/start`

### 1.2 Создай нового бота

1. Отправь команду: `/newbot`

2. BotFather спросит **имя бота** — это то, что видят пользователи.
   Напиши, например:
   ```
   Анна | NL Куратор
   ```
   или
   ```
   Миша - Здоровье и Бизнес
   ```

3. BotFather спросит **username бота** — это техническое имя, должно заканчиваться на `bot`.
   Напиши, например:
   ```
   anna_nl_curator_bot
   ```
   или
   ```
   misha_health_nl_bot
   ```

4. BotFather пришлёт **токен** — это длинная строка вида:
   ```
   7123456789:AAHx1234567890abcdefghijklmnop
   ```

   **ВАЖНО:** Сохрани этот токен! Никому не показывай, кроме администратора системы.

### 1.3 Настрой описание бота (опционально)

Отправь BotFather:
```
/setdescription
```

Выбери своего бота и напиши описание, например:
```
Привет! Я AI-помощник Анны по здоровью и бизнесу NL International.

Помогу подобрать продукты под твои цели и отвечу на вопросы.
```

### 1.4 Установи аватарку бота (опционально)

Отправь BotFather:
```
/setuserpic
```

Выбери своего бота и отправь фото (твоё фото или логотип).

## Шаг 2: Создание группы в Telegram

### 2.1 Создай группу

1. В Telegram нажми **меню** (три полоски) → **Новая группа**
2. Добавь хотя бы одного человека (можешь добавить администратора системы)
3. Назови группу, например: `NL | Анна | Контент`

### 2.2 Сделай группу супергруппой с топиками (рекомендуется)

1. Открой группу → **Настройки** (три точки вверху)
2. **Управление группой** → **Тип группы**
3. Включи **Темы** (Topics)

Топики позволяют разделить контент по категориям:
- Продукты
- Бизнес
- Истории успеха
- Новости

### 2.3 Добавь бота в группу

1. Открой группу → **Добавить участника**
2. Найди своего бота по username (например: @anna_nl_curator_bot)
3. Добавь его

### 2.4 Сделай бота администратором

1. Открой список участников группы
2. Найди бота → нажми на него
3. **Назначить администратором**
4. Включи права:
   - Изменение профиля группы
   - Удаление сообщений
   - Публикация сообщений
   - Управление темами (если есть топики)

### 2.5 Получи ID группы

**Способ 1: Через бота @getidsbot**
1. Добавь @getidsbot в группу
2. Он напишет ID группы (начинается с `-100...`)
3. Удали бота из группы после этого

**Способ 2: Через веб-версию**
1. Открой web.telegram.org
2. Зайди в группу
3. В адресной строке будет: `web.telegram.org/a/#-1001234567890`
4. ID группы = `-1001234567890`

## Шаг 3: Получение Yandex Cloud API

Это нужно для работы AI (генерация текста и картинок).

### 3.1 Регистрация в Yandex Cloud

1. Перейди на https://console.cloud.yandex.ru/
2. Войди через Яндекс ID (или создай аккаунт)
3. Привяжи карту для оплаты

### 3.2 Создание платёжного аккаунта

1. В консоли выбери **Создать платёжный аккаунт**
2. Выбери **Физическое лицо** или **ИП/ООО**
3. Привяжи карту
4. Пополни баланс на **500-1000 рублей** для начала

### 3.3 Создание каталога (Folder)

1. В левом меню: **Все сервисы** → **Resource Manager**
2. Нажми **Создать каталог**
3. Назови: `nl-bot` или `ai-curator`
4. Нажми **Создать**
5. Открой созданный каталог
6. **Скопируй Folder ID** — он в URL или на странице каталога

```
Пример Folder ID: b1g8skpblkos03malf3s
```

### 3.4 Создание сервисного аккаунта

1. В каталоге перейди: **Сервисные аккаунты** → **Создать сервисный аккаунт**
2. Имя: `ai-bot-account`
3. Роли — добавь обе:
   - `ai.languageModels.user` (для YandexGPT — генерация текста)
   - `ai.imageGeneration.user` (для YandexART — генерация картинок)
4. Нажми **Создать**
5. **Скопируй Service Account ID**

```
Пример Service Account ID: aje76dc7i20078podfrc
```

### 3.5 Создание API-ключа

1. Открой созданный сервисный аккаунт
2. Вверху нажми **Создать новый ключ** → **Создать авторизованный ключ**
3. Скачается JSON-файл с ключом

**Внутри файла будет:**
```json
{
  "id": "ajensd96tl0d2q9fqmp9",
  "service_account_id": "aje76dc7i20078podfrc",
  "created_at": "2026-01-21T10:00:00Z",
  "key_algorithm": "RSA_2048",
  "public_key": "-----BEGIN PUBLIC KEY-----\n...",
  "private_key": "-----BEGIN PRIVATE KEY-----\nMIIE...\n-----END PRIVATE KEY-----\n"
}
```

- `id` — это **Key ID**
- `private_key` — это **Private Key**

**Сохрани этот файл! Отправишь его администратору.**

## Шаг 4: Собери ссылки из NL

### 4.1 Войди в личный кабинет NL

1. Зайди на https://nlstar.com/ (или твой региональный сайт)
2. Авторизуйся

### 4.2 Найди свой ID партнёра

Обычно отображается в личном кабинете или в URL.
```
Пример: 12345678
```

### 4.3 Найди реферальную ссылку на магазин

1. В кабинете найди раздел **Реферальные ссылки** или **Пригласить**
2. Скопируй ссылку на магазин с твоим ID:
```
Пример: https://nlstar.com/shop/?ref=abc123
```

### 4.4 Найди ссылку на регистрацию партнёра

```
Пример: https://nlstar.com/ref/abc123/
```

## Шаг 5: Заполни анкету

Заполни Google-форму (ссылка от администратора) или отправь информацию напрямую:

### Данные для анкеты:

**1. Основное:**
- Твоё имя (как бот будет представляться)
- Телеграм для связи с клиентами (@username)
- Телефон (опционально)

**2. Telegram-бот:**
- Токен бота (из BotFather)
- Username бота (@...)

**3. Telegram-группа:**
- ID группы (начинается с -100...)
- Название группы

**4. Yandex Cloud:**
- Folder ID
- Service Account ID
- Key ID
- JSON-файл с ключом (прикрепить)

**5. Ссылки NL:**
- ID партнёра NL
- Ссылка на магазин
- Ссылка на регистрацию партнёра

**6. Профиль бота:**
- Специализация (выбери 1-2):
  - Похудение / контроль веса
  - Энергия / витамины
  - Бизнес / заработок
  - Красота / уход
  - Всё понемногу

- Твоя история (2-3 предложения):
  Почему ты в NL? Какие результаты у тебя или твоих клиентов?

- Стиль общения:
  - Дружелюбный, на "ты" (рекомендуется)
  - Более формальный, на "вы"

## Что будет дальше

1. **Администратор получает твою анкету** и проверяет данные
2. **Настраивает бота** на сервере (обычно в течение 24 часов)
3. **Отправляет тебе подтверждение** — бот готов к работе
4. **Проводится тест** — проверяем, что всё работает

## Частые вопросы

### Я не технарь, справлюсь?

Да! Инструкция пошаговая, занимает около 1 часа. Если что-то непонятно — пиши, поможем.

### Что если я что-то сделаю не так?

Ничего страшного. Все данные проверяются перед запуском. Если что-то не так — администратор напишет, что нужно исправить.

### Могу ли я изменить настройки потом?

Да, можно изменить: имя бота, специализацию, историю, ссылки. Просто напиши администратору.

### Сколько клиентов может обрабатывать бот?

Практически неограниченно. Бот работает на мощном сервере.

### А если Yandex Cloud заблокирует?

У каждого партнёра свой аккаунт Yandex — если один заблокируют (маловероятно), остальные продолжат работать.

### Как следить за статистикой?

В боте будут команды:
- `/my_stats` — твоя статистика (лиды, конверсии)
- `/my_leads` — список твоих лидов
